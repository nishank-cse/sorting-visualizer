## 🚀 Project Overview

This project visualizes various sorting algorithms in real-time and logs run history to a database. Users can:

✔ Generate random arrays  
✔ Visualize sorting algorithms such as:  
  • Bubble Sort  
  • Selection Sort  
  • Insertion Sort  
  • Quick Sort  
✔ View historical performance data  
✔ Save sorting runs to a database

The backend stores run records (algorithm, time taken, array size), which frontend displays as history.

## 📦 Features

- 🧠 **Interactive Sorting Visualization**  
Visualizes the step-by-step execution of sorting algorithms.

- 📊 **Run History Tracking**  
Backend API logs sorting runs in MongoDB, and the frontend displays history.

- 📈 **Responsive UI**  
Easy controls for array size, algorithm selection, and speed.
